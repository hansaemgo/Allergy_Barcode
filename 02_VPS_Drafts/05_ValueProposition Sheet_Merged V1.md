## 1. Executive Summary (요약)

- **핵심 타겟:**
  - **1차 (B2C):** 장보기 시간 부족과 교차오염 공포에 시달리는 **30대 워킹맘(김지윤)** & 복합 항원 관리로 인지 부하 한계에 달한 **다자녀 부모(최수안)**
  - **1차 (B2B):** 단 한 건의 배식 사고로 인한 소송 및 폐원 리스크를 100% 방어해야 하는 **프리미엄 어린이집 원장(박현진)**
  - **데이터 인프라 (Extreme):** 법적 사각지대에 놓여 제조사와 싸우는 **희귀 환우(유나비)**
- **핵심 Job:**
  - "이 간식을 아이에게 먹여도 되는지 고민 없이 단 0.5초 만에 안전 여부를 알고 싶다." (B2C)
  - "학부모의 컴플레인과 오배식 책임을 시스템적으로 원천 차단해 사업장을 방어하고 싶다." (B2B)
- **핵심 Value Proposition:**
  "B2C 학부모에게는 **정보 해독이 아닌 판단 자체를 대행해 주는 '0.5초 투명 렌즈'**를, B2B 기관장에게는 **소송과 파산을 물리적으로 막아주는 불변의 '디지털 행정 방어벽'**을 하나의 엔진으로 제공합니다."
- **MVP 포커스:**
  1) 0.5초 판별 스캐너(바코드/OCR) + 시각/햅틱 알람
  2) B2B 다중 원아 식단 매칭 관리자 대시보드 + 3중 원격 알림망
  3) 계정 스위칭 없는 다자녀(가족) 한 번에 필터 연산
  4) 희귀 성분 검증용 크라우드소싱 제보 위키

## 2. AOS–DOS Combined Matrix (기회 분석 및 전략적 예산 배분)

AOS(고통의 순수 크기)와 DOS(시장 수익성 보정치)를 결합하여, 단순히 아픈 곳이 아닌 **'돈이 되는 아픈 곳'에 마케팅과 개발 예산을 100% 최우선 투입**하기 위한 핵심 매트릭스입니다.

### 📈 페르소나별 전략적 우선순위 평가

| 페르소나 | AOS | MR | DOS | 전략적 해석 및 서사적 가치 부여 |
| :--- | :---: | :---: | :---: | :--- |
| **원장 박현진** (B2B) | 4.0 | 0.9 | **3.60 (1위)** | [시스템 방어자] '오배식에 의한 폐원 및 소송 리스크'를 짊어진 가장 지불 의향이 높은 타겟. B2C 확산의 필수적인 Lock-in 전진 기지. |
| **워킹맘 김지윤** (B2C) | 3.0 | 1.0 | **3.00 (2위)** | [맹목적 보호망] '판단의 무게'로 극도의 피로를 겪으나, 맘카페 기반의 막연한 공포를 쾌속 전파하는 바이럴 폭발력(MR=1.0) 1위 집단. |
| **환우 유나비** (Extreme) | 4.0 | 0.7 | **2.80 (3위)** | [사각지대 개척자] 국가가 방관하는 식약처 밖 정보의 희생자이나, 이들의 맹목적 분노를 크라우드소싱 시스템으로 흡수하여 독보적 '희귀 데이터 인프라'를 확보. |
| **영양교사 송미정** (B2G) | 3.0 | 0.8 | **2.40 (4위)** | B2G 조달 난이도로 후순위이나, NEIS 연동 자동화에 대한 열망이 매우 강력. |
| **조부모 신영숙** (B2C) | 3.0 | 0.6 | **1.80 (5위)** | 앱 설치 전환 난이도 극악하나, Zero-depth UI로 포용적 설계 완성. |
| **다자녀 최수안** (B2C) | 2.4 | 0.8 | **1.60 (6위)** | 자녀별 프로필 필터링 락인으로, 한 번 적응하면 타 앱으로 빠져나갈 수 없는 방패막 타겟. |

### 💰 DOS 기반 4구역 예산 배분 타겟팅

| 구역 | DOS 범위 | 대상 | 전략 |
| :--- | :--- | :--- | :--- |
| 🚨 **1차 PMF 폭발 (Cash-Cow)** | DOS ≥ 3.0 | **박현진**(3.6), **김지윤**(3.0) | 마케팅·영업 예산 100% 최우선 투입. 1년 차 SOM 전량 이 구역에서 발생. |
| 🛠️ **2차 신뢰 해자 (Moat)** | DOS 2.0~2.9 | **유나비**(2.8), **송미정**(2.4) | 직접 수익보다 크라우드소싱 DB 구축·B2G 레퍼런스 확보에 투자. |
| 📉 **3차 보조 확장** | DOS 1.0~1.9 | **신영숙**(1.8), **최수안**(1.6) | 다자녀 모드처럼 기존 유입자의 이탈을 막는 방어 기제/서브 기능 역할. |
| 🛑 **4차 타겟 폐기** | DOS < 1.0 | **강태민**, **정하늘** 등 | 기존 대안에 만족 → 초기 마케팅 예산 절대 투입 금지. |

## 6. 수익 구조 설계 (TAM-SAM-SOM & Monetization)

단순한 과금 체계를 넘어 강력한 **배상 책임 방어망과 락인(Lock-in)**을 무기로 즉각 현금을 창출합니다.

### 시장 규모

- **TAM (글로벌 Free-From 시장):** 약 58조 원 (비건, 포드맵, 대체 식품 전체 / CAGR 7.7%)
- **SAM (국내 안심 소비/보육 통제망):** 약 2,208억 원 (0-12세 유효 가구 + 약 3만 개 어린이집)
- **SOM (초기 1년 목표): 연간 최소 1.98억 원 달성 모델**
  - **B2C (연 9천만 원/SOM-1):** 월 4,900원(다자녀 플랜 구독) + 파트너 브랜드 맞춤 '안심 간식 커머스 중개 수수료 5%' (5,000가구 타겟)
  - **B2B (연 1.08억 원/SOM-3):** 300개 프리미엄 기관 선점. 월 3만~10만 원 라이선스 + '배상 책임 공제 보험 번들링 + 안심 보육 시설 인증 현판 발급' 복합 세일즈

### 수익 모델 구조

| 플랜 | 대상 | 내용 | 과금 |
| :--- | :--- | :--- | :--- |
| **Free** | B2C 신규 유저 | 일일 스캔 3회 제한 + 단일 프로필 | 무료 |
| **Family** | B2C 유료 구독 | 무제한 스캔 + 다자녀 N명 동시 필터 + 보호자 초대 연동 | 월 4,900원 |

| **School** | B2G 학교 | NEIS API 연동 + 담임 자동 발송 + 조달청 등록 | 연간 라이선스 (v2.0) |

## 7. 핵심 리스크, 가설 및 Contingency Plan

IR 돌파 및 실무의 맹점을 방어할 최우선 대비책 리스트입니다.

### 7.1 수익 / 시장 가설

| 핵심 가설 검증 목표 | 검증 평가 기준 | 실패 시 Pivot 대안 (Contingency Plan) |
| :--- | :--- | :--- |
| **B2B 원장은 과연 마케팅 무기가 된다고 결제할까?** | B2B 테스트 도입 기관 학부모 설문 시 도입 인지 60%, 긍정 영향 30% 이상 | 반응 저조 시 '보육 기관 인증 오프라인 현판 및 스티커' 등 실질적 가시화 도구 패키징 전면 무상 확대 제공 |
| **핵심 타겟인 맘카페 바이럴이 즉각 유입으로 전환될까?** | 단일 채널 온보딩 대비 CAC 비용 50% 이하 절감 확인 | 유입 더딜 시 소아청소년과 제휴 홍보지 배치 및 틱톡/릴스 숏폼 UGC 활용 우회 채널 확대 |
| **무료 하드리밋(스캔 3회 제한)이 정기 구독료로 넘어올까?** | 스캔 제한치별(3회/5회/10회) 분할 후, 최종 멤버십 전환율 8% 타겟 추적 | 하드리밋 거부 발생 시 제한 해제 대신 '가족 다중 연산', '커머스 리워드' 기반의 가치 잠금형 수익 모델 도입 |

### 7.2 최악의 운영(Operational) 위기 대응

| 치명적 리스크 상황 | 사전 방어 플랜 (Preventive) | 사후 대처 플랜 (Reactive) |
| :--- | :--- | :--- |
| **앱의 오판(DB 지연 등)으로 아이가 쇼크에 빠져 고소당함** | 이용 약관 가입 시 "본 정보는 보조 수단임" 강제 동의. 교차오염 위험성 있는 경우 극단적 보수 결과 송출(노란색 알람) | B2B 기관 플랜 영업 시 **"알러지 오배식 배상 책임 보험"** 결합 보장. 긴급 Push 공지 동시 발송. |
| **식약처 공공 API 서버 마비로 실시간 체질 대조 중단** | 주기적 로컬 미러링 스냅샷 저장 및 민간 대체재 복수 후보(CJ, 유통진흥원) 병렬 대기 처리 | 즉각 장애 자동 인식 후 캐시 DB로 스위칭 변환, 유저 인앱 팝업으로 현재 DB의 기준일자 알림 제공 |
| **일선 조리사/보육교사의 복잡한 태블릿 다루기 항명 탈주** | 기관용 모드는 터치 한 번 없는 거치형 패드로 자동 바코드 렌즈 송출 모드 강요 | 50% 미만 점유 기관의 경우 인적 재방문 교육 실행. 미사용 교사 식별 리포트를 기관장 관리 권한으로 위임 |

## 8. MVP 구현 상세 계획 (12개월 타임라인)

DOS 3.0 이상 집단의 생존과 지갑을 직결시킬 핵심 실험 타임라인입니다. Job-Feature Map(섹션 5)의 우선순위에 따라 Phase별로 구현합니다.

### Phase 1 (Month 1~3): MVP 개발 + 코어 DB 시동

- **구현 대상 Feature:** F1(0.5초 스캐너), F2(B2B Web), F3(다자녀 필터), F5(제보 위키 최소 기능)
- **비즈니스 목표:**
  - 식약처 공공데이터 API 연동 + 초기 100만 건 DB 구축
  - 0.5초 응답 속도 최적화 구현
  - 알레르기 전문 병원과의 의학 검수 제휴(MOU) 확보
  - 환우회 파워 유저 30인 시드 데이터 발굴 및 모집
- **가설 검증 잣대:**
  - 임계속도(0.5초) 미달 시 → 오프라인 로컬 캐시 스위칭 강화
  - 텍스트 vs 컬러 사용성 불일치 시 → 하이브리드 UI 즉각 분리 채비

### Phase 2 (Month 4~5): 파일럿(PoC) 무상 검증

- **구현 대상 Feature:** F4(3중 알림 통신망) 실제 기관 적용 테스트
- **비즈니스 목표:**
  - B2B 프리미엄 어린이집 30곳 무상 시범 도입 (배식 에러율, 교사 사용률, 탈주율 추적)
  - B2C 맘스홀릭 등 코어 커뮤니티 시드 배포 (NPS 및 이탈 구간 최적화)
  - 소아청소년과 제휴 홍보지 배치
- **가설 검증 잣대:**
  - 교사들의 스캔 완료율 50% 미만 추락 시 → 로그인 불요 및 터치제로 UX 전면 스위칭
  - 현장 재교육 무상 지원 실행

### Phase 3 (Month 6~8): 정식 런칭 및 유료 전환

- **구현 대상 Feature:** F6(시니어 모드), F7(커머스 연동) 추가 배포
- **비즈니스 목표:**
  - 테스트 30곳 유료 계정 전환 달성 (목표: 100곳)
  - 안심기관 인증 스티커/현판 동시 패키지 배포
  - B2C 커머스 탭 오픈, 추천/결제 수수료 파이프라인 활성화
  - B2C MAU 5,000가구 달성
- **가설 검증 잣대:**
  - Free 하드리밋 저항 심할 시 → 제한 완화 후 프리미엄 혜택(다중 연산 등) 중심으로 유료 모델 피벗

### Phase 4 (Month 9~12): 플라이휠 완성 & 스케일 업

- **비즈니스 목표:**
  - B2B 300곳 달성 → 연 1.08억 원 안정적 고정 매출 확보
  - B2C 커머스 수수료 수익 본격화 → 연 9,000만 원 달성
  - **SOM 1년 차 총 목표: 1.98억 원 ($150K) 달성**
  - 축적된 NLP 엔진 베이스를 활용해, **비건/포드맵/클린라벨/할랄 (58조 확장 인접 타겟)** 필터만 바꿔끼우는 Scale-out 착수 논의
- **가설 검증 잣대:**
  - 환우회 유입량 급증으로 인한 자체 DB 오염 우려 시 → 고객센터 캡처 인증 필수 채택
  - 고소/고발 리스크 시 → 임원배상책임(D&O) 보험 즉각 적용
